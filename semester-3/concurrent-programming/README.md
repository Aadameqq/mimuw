# Concurrent Programming (Programowanie współbieżne)

Basic problems and techniques of concurrent and distributed system programming. Classic concurrency problems. Synchronization mechanisms in a centralized model: semaphores and monitors. Synchronization methods in a distributed model, synchronous and asynchronous communication. Verification of concurrent program correctness. Selected distributed algorithms. Complexity measures of concurrent programs.
