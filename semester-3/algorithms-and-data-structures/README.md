# Algorithms and Data Structures (Algorytmy i struktury danych)

Design and analysis of algorithms. Overview of basic algorithms and data structures. Improving practical skills in designing and programming correct and efficient algorithms, and using ready-made libraries of algorithms and data structures.

**Key Topics Covered:**
*   **Analysis & Design:** Basic principles of algorithm analysis and methods for designing efficient algorithms.
*   **Data Structures:** Priority queues, dictionaries, text data structures, and the Union-Find problem.
*   **Core Algorithms:** Sorting, selection, and searching.
*   **Graph Theory:** Fundamental graph algorithms.
*   **Text Processing:** Pattern matching and text algorithms.
